Produits de la Mapserver Suite¶

Note

Depuis la publication de Mapserver 6.0, le projet Mapserver inclut une suite de produis Open Source, pour fournir un jeu complet d’outils pour la cartographie en ligne à la communauté. Le Comité de Gouvernance de Projet de Mapserver maintient tous ces produits sous la même dénomination Mapserver.

Note

Les développeurs souhaitant contribuer aux projets sur Github peuvent revoir les règles du guide Github de Mapserver

Coeur de Mapserver¶

Le code source du cœur de MapServer, écrit en C, et toujours un des plus rapide et des plus configurable des moteurs cartographiques en ligne dans le monde.

  • Téléchargement

  • Accueil Github

MapCache¶

Depuis la version de 6.0 de MapServer, MapServer inclut aussi des fonctionnalités de mise en cache très puissantes par l’intermédiaire du projet MapCache.

  • Accueil de la documentation

  • Téléchargement

  • Accueil Github

TinyOWS¶

Depuis la version de 6.0 de MapServer, MapServer a aussi la capacité d’effectuer des requêtes transactionnelles (édition géographique en ligne) via la spécification WFS-T grâce au projet TinyOWS.
